# avms_project
Apartment_visitor_management_system_Web_project
The Apartment Visitor Management System (AVMS) is a web-based solution designed to modernize and digitize visitor tracking within residential complexes.
Key Features:

    🔐 CRUD Operations – Create, Read, Update, Delete visitor records and passes

    📊 Real-time Dashboard – Daily, weekly, and total visitor statistics

    🏷️ Category Management – Classify visitors (e.g., Milkman, Guest, Delivery)

    🎟️ Digital Pass Generation – Time‑based entry passes with validity

    ✏️ Outing Remarks – Track exit time and notes for each visitor

    🗂️ Date‑Filtered Reports – Historical record retrieval
    
Tech Stack:
PHP | MySQL | HTML5 | CSS3 | JavaScript | XAMPP | VS Code
    Query successful

### System Implementation

The system utilizes a modular architecture to handle distinct operations:

    Dashboard Module: Displays real-time visitor statistics.

    Visitor & Category Management: Manages detailed logs and service types (e.g., Milkman, Newspaper).

    Reporting: Generates historical data reports using SQL date-filtering logic.
